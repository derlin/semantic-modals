semanticModal service demo

Here are some examples, with the options and the result. All pre use the function call ModalService.showModal(options);.
A closable confirm modal. 
var options = {
    title: "confirm",
    text: "are you sure ?",
    positive: "yep",
    negative: "cancel",
    cancelable : true
};
A confirm modal with html content, basic style and semantic icon. 
var options = {
    title   : "confirm",
    html    : "
are you absolutely sure ?
All its SLS will be deleted as well.
",
    positive: "proceed",
    negative: "cancel",
    icon    : "warning sign orange",
    basic   : true,
};
A basic modal with angular bindings, non cancelable. 
var options = {
    title: "confirm",
    html: "are you sure you want to delete {{inputs.sensor.name}} ({{inputs.sensor.id}} ) ?",
    positive: "yes",
    basic: true,
    inputs     : {
        sensor: {id: 3, name: "sensor-XXXX"}
    },
    cancelable : false
} );
A modal using a custom content. Internally, semanticModal uses the ng-include directive. 
var options = {
    title: "html include",
    htmlInclude: "partial/editModal_content.html",
    positive: "do it!",
    cancelable : false
} );
a completely custom content. You have access to ctrl.inputs and can use closeModal(true|false) to close the modal and set the return status. 
var options = {
    title: "html include",
    templateUrl: "semanticModalTemplate.html",
    cancelable : false
} );
